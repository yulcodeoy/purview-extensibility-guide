# Microsoft Purview Extensibility Guide

A personal, **unofficial** working reference on Microsoft Purview extensibility surfaces — APIs, SDKs, PowerShell, connectors, portals — organized by partner-recognizable scenarios, workloads, and modalities.

**Live site:** <https://yulcodeoy.github.io/purview-extensibility-guide/>

This is a personal field guide for partners, customers, and developers building on Purview. **It is not an official Microsoft publication.** Portions of this catalog were drafted with AI assistance and are continuously reviewed against the canonical Microsoft Learn documentation that every entry links to. Always reconfirm specifics (rate limits, scopes, GA status, partner program details) against the linked Learn page before relying on them.
